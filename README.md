# Code Packer

**让 AI 真正读懂你的项目。**

这是一个专为 LLM (ChatGPT, Claude, Gemini) 辅助编程设计的**上下文构建工具**。它能将你的整个工程文件夹瞬间转化为 AI 易读的、结构化的单个 Prompt 文件。
  
![](https://cdn.jsdelivr.net/gh/dozybot001/blogimage/20251209210737996.png)

## 🎯 为什么你需要这个工具？

在利用 AI 写代码时，你是否经历过以下**痛点**？

| 😫 以前的痛点 | ⚡️ 本工具的解决方案 |
| :--- | :--- |
| **手动搬运繁琐**：需要在 IDE 和浏览器之间反复切换，一个个复制粘贴文件。 | **一键打包**：拖拽整个文件夹，毫秒级生成所有代码内容的聚合文本。 |
| **上下文丢失**：直接粘贴代码片段，AI 不知道文件路径，不知道引用关系。 | **结构保留**：自动生成 ASCII 目录树，并在每段代码前标注文件路径，让 AI 拥有“全局视野”。 |
| **Token 浪费**：不小心复制了 `node_modules` 或 `.git`，瞬间撑爆上下文窗口。 | **智能过滤**：内置主流语言（JS, Python, Go, Flutter 等）的忽略规则，只保留核心逻辑代码。 |
| **隐私焦虑**：不敢将公司项目的代码上传到第三方“代码分析网站”。 | **绝对隐私**：纯前端运行（Vanilla JS），代码处理完全在你的浏览器本地完成，**绝不上传服务器**。 |

## 💡 核心价值

### 1\. 为 AI 提供“上帝视角”

简单的复制粘贴只能让 AI 看到局部。本工具生成的 Prompt 包含**完整的项目目录树**和**带路径标记的代码块**。

> **结果：** AI 不再只会写孤立的函数，而是能理解模块间的依赖，给出更精准的架构级建议。

### 2\. 极致的 Token 效率

并不是所有文件都对编程有帮助。本工具会自动剔除：

  * **依赖库** (`node_modules`, `venv`, `target`)
  * **版本控制** (`.git`)
  * **二进制/媒体文件** (`.png`, `.lock`, `.exe`)

> **结果：** 将原本 100k Token 的混乱项目，精简为 10k Token 的高价值上下文。

### 3\. 零门槛，零风险

  * **无需安装**：没有 Python 依赖，没有 npm install，双击 HTML 文件即可使用。
  * **安全可控**：代码不出本机，甚至断网也能使用。

## 🚀 快速开始

### 在线使用 (GitHub Pages)

[👉 点击这里立即使用](https://dozybot001.github.io/code_packer/)

### 本地运行

1.  克隆或下载本仓库。
2.  双击打开 `index.html`。
3.  将你的项目文件夹拖入上传区。
4.  点击下载或复制生成的 Prompt。

## 📝 最佳实践

**如何向 AI 提问？**
将本工具生成的内容发送给 AI 后，推荐使用以下 Prompt 开头：

> "这是我项目的完整代码上下文。请根据目录结构和文件内容，理解项目架构，然后[你的具体需求，例如：帮我重构 api\_service 模块 / 解释数据流向]..."
